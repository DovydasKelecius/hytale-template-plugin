# com.hypixel.hytale.builtin

This package contains built-in game features and functionality.

## Sub-packages

The `com.hypixel.hytale.builtin` package is organized into the following sub-packages, each of which represents a specific game feature:

*   `adventure`: Contains features related to adventure mode, such as quests, farming, and reputation.
*   `ambience`: Contains features related to ambient sounds and music.
*   `asseteditor`: Contains the in-game asset editor.
*   `beds`: Contains features related to beds and sleeping.
*   `blockphysics`: Contains features related to block physics.
*   `blockspawner`: Contains features related to block spawners.
*   `blocktick`: Contains features related to block ticking.
*   `buildertools`: Contains tools for building and editing the world.
*   `commandmacro`: Contains features related to command macros.
*   `crafting`: Contains features related to crafting.
*   `creativehub`: Contains features related to the creative hub.
*   `crouchslide`: Contains features related to crouching and sliding.
*   `deployables`: Contains features related to deployable items.
*   `fluid`: Contains features related to fluids.
*   `hytalegenerator`: Contains the world generator.
*   `instances`: Contains features related to instanced worlds.
*   `landiscovery`: Contains features related to LAN server discovery.
*   `mantling`: Contains features related to mantling.
*   `model`: Contains features related to models.
*   `mounts`: Contains features related to mounts.
*   `npccombatactionevaluator`: Contains the NPC combat action evaluator.
*   `npceditor`: Contains the in-game NPC editor.
*   `parkour`: Contains features related to parkour.
*   `path`: Contains features related to pathfinding.
*   `portals`: Contains features related to portals.
*   `safetyroll`: Contains features related to safety rolling.
*   `sprintforce`: Contains features related to sprinting.
*   `tagset`: Contains features related to tag sets.
*   `teleport`: Contains features related to teleportation.
*   `weather`: Contains features related to weather.
*   `worldgen`: Contains features related to world generation.
