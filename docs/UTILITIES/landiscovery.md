# com.hypixel.hytale.builtin.landiscovery

This package contains features related to LAN server discovery.

## Overview

The `landiscovery` package provides a system for discovering Hytale servers on the local area network (LAN).

### Key Classes

*   `LANDiscoveryPlugin`: The main plugin class for the LAN discovery feature.
*   `LANDiscoveryThread`: A thread that broadcasts and listens for LAN server announcements.
*   `LANDiscoveryCommand`: A command that allows players to manage LAN discovery.
