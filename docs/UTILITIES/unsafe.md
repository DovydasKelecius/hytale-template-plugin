# com.hypixel.hytale.unsafe

This package contains unsafe utility classes.

## Overview

The `unsafe` package contains utility classes that provide direct access to low-level memory operations, typically exposed by `sun.misc.Unsafe`. These classes are used for highly optimized, performance-critical operations that bypass normal Java safety checks. Use of this package implies a deep understanding of memory management and potential risks.

### Key Classes

*   `UnsafeUtil`: A utility class that provides access to `sun.misc.Unsafe` functionalities.

### Sub-packages

This package currently does not contain any further sub-packages.
