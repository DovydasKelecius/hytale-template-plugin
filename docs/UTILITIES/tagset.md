# com.hypixel.hytale.builtin.tagset

This package contains features related to tag sets.

## Overview

The `tagset` package provides a system for managing tag sets, which are collections of tags that can be applied to various game elements.

### Key Classes

*   `TagSetPlugin`: The main plugin class for the tag set feature.
*   `TagSet`: A class that represents a tag set.
*   `TagSetLookupTable`: A lookup table for tag sets.

### Sub-packages

*   `config`: Contains configuration related to tag sets.
