# com.hypixel.hytale.builtin.path

This package contains features related to pathfinding.

## Overview

The `path` package provides a system for pathfinding for NPCs and other entities. It includes a plugin class for the feature and classes for defining paths, waypoints, and commands.

### Key Classes

*   `PathPlugin`: The main plugin class for the pathfinding feature.
*   `PrefabPathCollection`: A collection of prefab paths.
*   `WorldPathBuilder`: A builder for creating world paths.
*   `WorldPathData`: Data for a world path.
*   `PrefabPathCommand`: A command that allows players to manage prefab paths.

### Sub-packages

*   `commands`: Contains commands related to pathfinding.
*   `entities`: Contains entities related to pathfinding.
*   `path`: Contains classes for defining paths.
*   `waypoint`: Contains classes for defining waypoints.
