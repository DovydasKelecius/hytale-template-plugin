# com.hypixel.hytale.builtin.creativehub

This package contains features related to the creative hub.

## Overview

The `creativehub` package provides a system for managing the creative hub, which is a central place for players to showcase their creations and to access creative mode tools.

### Key Classes

*   `CreativeHubPlugin`: The main plugin class for the creative hub feature.
*   `HubCommand`: A command that allows players to teleport to the creative hub.
*   `HubPortalInteraction`: An interaction that allows players to use portals to travel to and from the creative hub.
*   `CreativeHubWorldConfig`: A config that defines the creative hub world.

### Sub-packages

*   `command`: Contains the `HubCommand` class.
*   `config`: Contains the `CreativeHubWorldConfig` class.
*   `interactions`: Contains the `HubPortalInteraction` class.
