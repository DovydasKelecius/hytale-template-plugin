# com.hypixel.hytale.builtin.buildertools

This package contains tools for building and editing the world.

## Overview

The `buildertools` package provides a powerful set of tools for building and editing the world in real-time. It includes a plugin class for the feature and classes for managing the builder tools UI and functionality.

### Key Classes

*   `BuilderToolsPlugin`: The main plugin class for the builder tools feature.
*   `BuilderToolsPacketHandler`: A packet handler for the builder tools.
*   `BuilderToolsSystems`: Contains the systems that manage the builder tools.
*   `BuilderToolsUserData`: A component that stores user data for the builder tools.

### Sub-packages

*   `commands`: Contains commands for using the builder tools.
*   `imageimport`: Contains classes for importing images into the world.
*   `interactions`: Contains interactions related to the builder tools.
*   `objimport`: Contains classes for importing OBJ files into the world.
*   `prefabeditor`: Contains the in-game prefab editor.
*   `prefablist`: Contains UI for listing and managing prefabs.
*   `scriptedbrushes`: Contains classes for creating and using scripted brushes.
*   `snapshot`: Contains classes for creating and restoring snapshots of the world.
*   `tooloperations`: Contains the different operations that can be performed with the builder tools.
*   `utils`: Contains utility classes for the builder tools.
