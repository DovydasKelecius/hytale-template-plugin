# com.hypixel.hytale.builtin.commandmacro

This package contains features related to command macros.

## Overview

The `commandmacro` package provides a system for creating and managing command macros. It includes a plugin class for the feature and classes for defining and executing macros.

### Key Classes

*   `MacroCommandPlugin`: The main plugin class for the command macro feature.
*   `MacroCommandBuilder`: A builder class for creating command macros.
*   `MacroCommandBase`: A base class for command macros.
*   `MacroCommandParameter`: A class that represents a parameter for a command macro.
*   `MacroCommandReplacement`: A class that represents a replacement for a command macro.
*   `EchoCommand`: An example command that echoes a message to the player.
*   `WaitCommand`: An example command that waits for a specified amount of time.
