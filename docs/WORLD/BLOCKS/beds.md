# com.hypixel.hytale.builtin.beds

This package contains features related to beds and sleeping.

## Overview

The `beds` package provides a system for players to sleep in beds, which can be used to skip the night and set the player's spawn point.

### Key Classes

*   `BedsPlugin`: The main plugin class for the beds feature.
*   `BedInteraction`: An interaction that allows players to use beds.
*   `PlayerSleep`: A component that tracks a player's sleep state.
*   `WorldSleep`: A resource that tracks the sleep state of the world.

### Sub-packages

*   `interactions`: Contains the `BedInteraction` class.
*   `respawn`: Contains UI pages related to setting the player's spawn point.
*   `sleep`: Contains components, resources, and systems related to sleeping.
