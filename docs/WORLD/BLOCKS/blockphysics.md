# com.hypixel.hytale.builtin.blockphysics

This package contains features related to block physics.

## Overview

The `blockphysics` package provides a system for simulating block physics, such as gravity and block falling.

### Key Classes

*   `BlockPhysicsPlugin`: The main plugin class for the block physics feature.
*   `BlockPhysicsSystems`: Contains the systems that manage block physics.
*   `BlockPhysicsUtil`: Contains utility methods for the block physics system.
