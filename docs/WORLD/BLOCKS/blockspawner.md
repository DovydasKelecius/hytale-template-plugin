# com.hypixel.hytale.builtin.blockspawner

This package contains features related to block spawners.

## Overview

The `blockspawner` package provides a system for creating and managing block spawners. It includes a plugin class for the feature and classes for defining block spawner assets and commands.

### Key Classes

*   `BlockSpawnerPlugin`: The main plugin class for the block spawner feature.
*   `BlockSpawner`: A component that represents a block spawner.
*   `BlockSpawnerCommand`: A command that allows players to manage block spawners.
*   `BlockSpawnerTable`: A table that defines the blocks that can be spawned by a block spawner.

### Sub-packages

*   `command`: Contains the `BlockSpawnerCommand` class.
*   `state`: Contains the `BlockSpawner` component.
