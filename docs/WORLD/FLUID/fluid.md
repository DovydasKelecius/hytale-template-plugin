# com.hypixel.hytale.builtin.fluid

This package contains features related to fluids.

## Overview

The `fluid` package provides a system for simulating fluids, such as water and lava. It includes a plugin class for the feature and classes for defining fluid assets and behavior.

### Key Classes

*   `FluidPlugin`: The main plugin class for the fluid feature.
*   `FluidState`: A component that represents a fluid block.
*   `FluidSystems`: Contains the systems that manage fluids.
*   `FluidCommand`: A command that allows players to manage fluids.
