# com.hypixel.hytale.builtin.worldgen

This package contains features related to world generation.

## Overview

The `worldgen` package provides a system for generating the world of Hytale. This package is specifically for built-in world generation features, while the `hytalegenerator` package contains the core world generation logic.

### Key Classes

*   `WorldGenPlugin`: The main plugin class for the world generation feature.

### Sub-packages

This package currently does not contain any further sub-packages.
