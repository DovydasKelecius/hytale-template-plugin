# com.hypixel.hytale.builtin.npccombatactionevaluator

This package contains the NPC combat action evaluator.

## Overview

The `npccombatactionevaluator` package provides a system for evaluating NPC combat actions. This is used to determine how NPCs respond in combat situations.

### Key Classes

*   `NPCCombatActionEvaluatorPlugin`: The main plugin class for the NPC combat action evaluator feature.
*   `CombatActionEvaluatorSystems`: Contains the systems that manage NPC combat action evaluation.
*   `CombatBalanceAsset`: An asset that defines combat balance.
*   `CombatActionEvaluatorConfig`: A config that defines the combat action evaluator.

### Sub-packages

*   `conditions`: Contains classes that define conditions for combat actions.
*   `config`: Contains asset definitions for combat.
*   `corecomponents`: Contains core components for NPC combat.
*   `evaluator`: Contains classes for evaluating combat actions.
*   `memory`: Contains classes related to NPC combat memory.
