# com.hypixel.hytale.builtin.npceditor

This package contains the in-game NPC editor.

## Overview

The `npceditor` package provides a system for editing NPCs in real-time. It includes a plugin class for the feature and classes for managing the NPC editor UI and functionality.

### Key Classes

*   `NPCEditorPlugin`: The main plugin class for the NPC editor feature.
*   `NPCRoleAssetTypeHandler`: A class for handling NPC role assets.
