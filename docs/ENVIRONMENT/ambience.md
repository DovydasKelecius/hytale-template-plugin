# com.hypixel.hytale.builtin.ambience

This package contains features related to ambient sounds and music.

## Overview

The `ambience` package provides a system for managing ambient sounds and music. It includes a plugin class for the feature and classes for defining and managing ambient emitters.

### Key Classes

*   `AmbiencePlugin`: The main plugin class for the ambience feature.
*   `AmbienceCommands`: A set of commands for managing ambience.
*   `AmbienceTracker`: A component that tracks the ambient sounds and music for a player.
*   `AmbientEmitterComponent`: A component that represents an ambient sound emitter.
*   `AmbienceResource`: A resource that stores all ambience data.
*   `AmbientEmitterSystems`: A set of systems that manage ambient emitters.
*   `ForcedMusicSystems`: A set of systems that manage forced music.

### Sub-packages

*   `commands`: Contains the `AmbienceCommands` class.
*   `components`: Contains the `AmbienceTracker` and `AmbientEmitterComponent` components.
*   `resources`: Contains the `AmbienceResource` class.
*   `systems`: Contains the `AmbientEmitterSystems` and `ForcedMusicSystems` classes.
