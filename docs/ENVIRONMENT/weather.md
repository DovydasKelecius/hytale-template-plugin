# com.hypixel.hytale.builtin.weather

This package contains features related to weather.

## Overview

The `weather` package provides a system for managing weather conditions in the game. It includes a plugin class for the feature and classes for defining weather assets and commands.

### Key Classes

*   `WeatherPlugin`: The main plugin class for the weather feature.
*   `WeatherCommand`: A command that allows players to manage weather.
*   `WeatherTracker`: A component that tracks the weather for a player.
*   `WeatherResource`: A resource that stores all weather data.
*   `WeatherSystem`: A system that manages weather.

### Sub-packages

*   `commands`: Contains the `WeatherCommand` class.
*   `components`: Contains the `WeatherTracker` component.
*   `resources`: Contains the `WeatherResource` class.
*   `systems`: Contains the `WeatherSystem` class.
