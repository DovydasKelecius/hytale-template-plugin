# com.hypixel.hytale.builtin.parkour

This package contains features related to parkour.

## Overview

The `parkour` package provides a system for players to engage in parkour activities. It includes a plugin class for the feature and classes for defining parkour checkpoints and commands.

### Key Classes

*   `ParkourPlugin`: The main plugin class for the parkour feature.
*   `ParkourCheckpoint`: A component that represents a parkour checkpoint.
*   `ParkourCommand`: A command that allows players to manage parkour.
*   `ParkourCheckpointSystems`: Contains systems that manage parkour checkpoints.

### Sub-packages

*   `commands`: Contains commands related to parkour.
