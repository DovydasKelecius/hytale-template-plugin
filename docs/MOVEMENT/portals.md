# com.hypixel.hytale.builtin.portals

This package contains features related to portals.

## Overview

The `portals` package provides a system for creating and managing portals, which can be used to travel between different areas of the world or to other dimensions.

### Key Classes

*   `PortalsPlugin`: The main plugin class for the portals feature.
*   `PortalDeviceConfig`: A config that defines a portal device.
*   `PortalDevice`: A component that represents a portal device.
*   `EnterPortalInteraction`: An interaction that allows players to enter portals.
*   `FragmentCommands`: Commands related to portal fragments.
*   `PortalWorld`: A resource that stores portal world data.

### Sub-packages

*   `commands`: Contains commands related to portals.
*   `components`: Contains components related to portals.
*   `integrations`: Contains integrations related to portals.
*   `interactions`: Contains interactions related to portals.
*   `resources`: Contains resources related to portals.
*   `systems`: Contains systems that manage portals.
*   `ui`: Contains UI related to portals.
*   `utils`: Contains utility classes for portals.
