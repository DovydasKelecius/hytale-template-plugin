# com.hypixel.hytale.builtin.teleport

This package contains features related to teleportation.

## Overview

The `teleport` package provides a system for players to teleport to various locations. It includes a plugin class for the feature and classes for defining warps and commands.

### Key Classes

*   `TeleportPlugin`: The main plugin class for the teleport feature.
*   `Warp`: A class that represents a warp point.
*   `WarpCommand`: A command that allows players to manage warps.
*   `TeleportHistory`: A component that tracks a player's teleport history.

### Sub-packages

*   `commands`: Contains commands related to teleportation.
*   `components`: Contains the `TeleportHistory` component.
