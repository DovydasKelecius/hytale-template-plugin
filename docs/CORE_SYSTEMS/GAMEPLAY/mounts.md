# com.hypixel.hytale.builtin.mounts

This package contains features related to mounts.

## Overview

The `mounts` package provides a system for players to mount entities, such as horses and minecarts. It includes a plugin class for the feature and classes for managing mount components and interactions.

### Key Classes

*   `MountPlugin`: The main plugin class for the mounts feature.
*   `BlockMountAPI`: An API for mounting blocks.
*   `BlockMountComponent`: A component that allows a block to be mounted.
*   `MountedComponent`: A component that indicates an entity is mounted.
*   `MountCommand`: A command that allows players to mount and dismount entities.
*   `MountInteraction`: An interaction that allows players to mount entities.

### Sub-packages

*   `commands`: Contains commands related to mounts.
*   `interactions`: Contains interactions related to mounts.
*   `minecart`: Contains classes related to minecarts.
*   `npc`: Contains classes related to NPC mounts.
