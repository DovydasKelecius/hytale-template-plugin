# com.hypixel.hytale.builtin.instances

This package contains features related to instanced worlds.

## Overview

The `instances` package provides a system for creating and managing instanced worlds, which are temporary, isolated copies of a world that players can enter to complete quests or other activities.

### Key Classes

*   `InstancesPlugin`: The main plugin class for the instances feature.
*   `InstanceValidator`: A validator for instances.
*   `ConfigurableInstanceBlock`: A block that can be configured to create an instance.
*   `InstanceBlock`: A block that creates an instance.
*   `InstanceEditCopyCommand`: A command that copies an instance.
*   `ExitInstanceInteraction`: An interaction that allows players to exit an instance.

### Sub-packages

*   `blocks`: Contains classes related to instance blocks.
*   `command`: Contains commands for managing instances.
*   `config`: Contains asset definitions for instances.
*   `event`: Contains events related to instances.
*   `interactions`: Contains interactions related to instances.
*   `page`: Contains UI pages related to instances.
*   `removal`: Contains classes related to instance removal.
