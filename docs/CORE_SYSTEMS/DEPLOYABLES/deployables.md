# com.hypixel.hytale.builtin.deployables

This package contains features related to deployable items.

## Overview

The `deployables` package provides a system for creating and managing deployable items, such as turrets and traps. It includes a plugin class for the feature and classes for defining deployable assets and behavior.

### Key Classes

*   `DeployablesPlugin`: The main plugin class for the deployables feature.
*   `DeployableComponent`: A component that represents a deployable item.
*   `DeployableConfig`: An asset that defines a deployable item.
*   `SpawnDeployableFromRaycastInteraction`: An interaction that allows players to deploy items.

### Sub-packages

*   `component`: Contains components related to deployable items.
*   `config`: Contains asset definitions for deployable items.
*   `interaction`: Contains interactions related to deployable items.
*   `system`: Contains systems that manage deployable items.
