# com.hypixel.hytale

This package is the core of the Hytale server and contains all the game logic and server functionality.

## Sub-packages

The `com.hypixel.hytale` package is organized into the following sub-packages:

*   `assetstore`: Contains classes for managing and storing game assets.
*   `builtin`: Contains built-in game features and functionality.
*   `codec`: Contains classes for encoding and decoding data.
*   `common`: Contains common utility classes.
*   `component`: Contains the entity-component system (ECS) framework.
*   `event`: Contains the event bus and event classes.
*   `function`: Contains functional interfaces.
*   `logger`: Contains logging-related classes.
*   `math`: Contains math-related utility classes.
*   `metrics`: Contains classes for collecting and reporting server metrics.
*   `plugin`: Contains the plugin management system.
*   `procedurallib`: Contains classes for procedural generation.
*   `protocol`: Contains the network protocol and packet classes.
*   `registry`: Contains classes for registering game objects.
*   `server`: Contains the core server implementation.
*   `sneakythrow`: Contains a utility class for throwing checked exceptions without declaring them.
*   `storage`: Contains classes for data storage.
*   `unsafe`: Contains unsafe utility classes.
