# com.hypixel.hytale.builtin.model

This package contains features related to models.

## Overview

The `model` package provides a system for managing 3D models in the game. It includes a plugin class for the feature and classes for managing model assets and commands.

### Key Classes

*   `ModelPlugin`: The main plugin class for the model feature.
*   `ModelCommand`: A command that allows players to manage models.
*   `ChangeModelPage`: A UI page that allows players to change models.

### Sub-packages

*   `commands`: Contains the `ModelCommand` class.
*   `pages`: Contains the `ChangeModelPage` class.
