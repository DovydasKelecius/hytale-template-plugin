# Hytale Server Documentation

This documentation provides a high-level overview of the Hytale Server project structure.

## Project Structure

The project is organized into the following main packages:

*   [`com.hypixel.fastutil`](fastutil.md): Contains utility classes for primitive types, based on the fastutil library.
*   [`com.hypixel.hytale`](hytale.md): The core of the Hytale server, containing all the game logic and server functionality.

For more detailed information about each package, please refer to the corresponding documentation file.
