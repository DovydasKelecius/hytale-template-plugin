# com.hypixel.hytale.builtin.adventure

This package contains features related to adventure mode, such as quests, farming, and reputation.

## Sub-packages

The `com.hypixel.hytale.builtin.adventure` package is organized into the following sub-packages:

*   `camera`: Contains classes related to camera effects and manipulation in adventure mode.
*   `farming`: Implements the farming system, including crop growth and harvesting.
*   `memories`: Manages player memories and related gameplay mechanics.
*   `npcobjectives`: Defines objectives and tasks for NPCs.
*   `npcreputation`: Handles reputation systems for NPCs.
*   `npcshop`: Contains functionality for NPC-driven shops.
*   `objectivereputation`: Manages reputation changes based on objective completion.
*   `objectives`: The core quest and objective system.
*   `objectiveshop`: Implements shops where objectives can be purchased or turned in.
*   `reputation`: A general-purpose reputation system.
*   `shop`: A general-purpose shop system.
*   `shopreputation`: Manages reputation changes based on shop interactions.
*   `stash`: Implements a personal stash for players.
*   `teleporter`: Contains the functionality for teleporters.
*   `worldlocationcondition`: Defines conditions based on world locations.
