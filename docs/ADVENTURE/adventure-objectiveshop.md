# com.hypixel.hytale.builtin.adventure.objectiveshop

This package implements shops where objectives can be purchased or turned in.

## Overview

The `objectiveshop` package provides a system for creating shops that are related to objectives. It includes a plugin class for the feature and classes for defining objective shop interactions.

### Key Classes

*   `ObjectiveShopPlugin`: The main plugin class for the objective shop feature.
*   `CanStartObjectiveRequirement`: A requirement that checks if a player can start an objective.
*   `StartObjectiveInteraction`: An interaction that starts an objective for a player.
