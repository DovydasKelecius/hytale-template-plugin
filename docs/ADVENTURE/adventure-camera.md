# com.hypixel.hytale.builtin.adventure.camera

This package contains classes related to camera effects and manipulation in adventure mode.

## Overview

The `camera` package provides the functionality for creating and managing camera effects, such as camera shake and view bobbing. It also includes a command for triggering these effects.

### Key Classes

*   `CameraPlugin`: The main plugin class for the camera feature.
*   `CameraEffectCommand`: A command that allows players to trigger camera effects.
*   `CameraShakeInteraction`: An interaction that can be used to trigger a camera shake.
*   `CameraEffectSystem`: A system that manages the application of camera effects to players.

### Sub-packages

*   `asset`: Contains asset definitions for camera effects.
*   `command`: Contains the `CameraEffectCommand` class.
*   `interaction`: Contains the `CameraShakeInteraction` class.
*   `system`: Contains the `CameraEffectSystem` class.
