# com.hypixel.hytale.builtin.adventure.teleporter

This package contains the functionality for teleporters.

## Overview

The `teleporter` package provides a system for creating and managing teleporters. It includes a plugin class for the feature and classes for defining teleporter assets and UI.

### Key Classes

*   `TeleporterPlugin`: The main plugin class for the teleporter feature.
*   `Teleporter`: A component that represents a teleporter.
*   `TeleporterInteraction`: An interaction that allows players to use teleporters.
*   `TeleporterSettingsPage`: A UI page that displays teleporter settings.

### Sub-packages

*   `component`: Contains the `Teleporter` component.
*   `interaction`: Contains the `TeleporterInteraction` class.
*   `page`: Contains the `TeleporterSettingsPage` class.
*   `system`: Contains systems that manage teleporters.
*   `util`: Contains utility classes for teleporters.
