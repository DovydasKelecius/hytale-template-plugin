# com.hypixel.hytale.builtin.adventure.npcreputation

This package handles reputation systems for NPCs.

## Overview

The `npcreputation` package provides a system for managing NPC reputation. This can be used to create factions and other social systems in the game.

### Key Classes

*   `NPCReputationPlugin`: The main plugin class for the NPC reputation feature.
*   `NPCReputationHolderSystem`: A system that manages the reputation of NPCs.
*   `ReputationAttitudeSystem`: A system that determines the attitude of an NPC towards a player based on their reputation.
