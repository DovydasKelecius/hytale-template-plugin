# com.hypixel.hytale.builtin.adventure.stash

This package implements a personal stash for players.

## Overview

The `stash` package provides a system for players to store their items in a personal stash. It includes a plugin class for the feature and a gameplay config.

### Key Classes

*   `StashPlugin`: The main plugin class for the stash feature.
*   `StashGameplayConfig`: A gameplay config for the stash feature.
