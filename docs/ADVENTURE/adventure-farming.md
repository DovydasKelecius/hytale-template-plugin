# com.hypixel.hytale.builtin.adventure.farming

This package implements the farming system, including crop growth and harvesting.

## Overview

The `farming` package provides the functionality for players to grow crops and raise animals. It includes systems for managing plant growth, animal breeding, and resource collection.

### Key Classes

*   `FarmingPlugin`: The main plugin class for the farming feature.
*   `FarmingSystems`: Contains the systems that manage the farming gameplay.
*   `FarmingUtil`: Contains utility methods for the farming system.
*   `HarvestCropInteraction`: An interaction that allows players to harvest crops.
*   `UseWateringCanInteraction`: An interaction that allows players to water crops.

### Sub-packages

*   `component`: Contains components related to farming.
*   `config`: Contains asset definitions for farming.
*   `interactions`: Contains interactions related to farming.
*   `states`: Contains block states related to farming.
