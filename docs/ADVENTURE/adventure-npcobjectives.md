# com.hypixel.hytale.builtin.adventure.npcobjectives

This package defines objectives and tasks for NPCs.

## Overview

The `npcobjectives` package provides a system for creating and managing objectives for NPCs. These objectives can be used to create quests and other interactive content.

### Key Classes

*   `NPCObjectivesPlugin`: The main plugin class for the NPC objectives feature.
*   `BountyObjectiveTaskAsset`: An asset that defines a bounty objective.
*   `KillObjectiveTaskAsset`: An asset that defines a kill objective.
*   `ActionStartObjective`: An action that can be used to start an objective for an NPC.
*   `SensorHasTask`: A sensor that can be used to check if an NPC has a specific task.

### Sub-packages

*   `assets`: Contains asset definitions for NPC objectives.
*   `npc`: Contains classes related to NPC behavior and objectives.
*   `resources`: Contains resources related to NPC objectives.
*   `systems`: Contains systems that manage NPC objectives.
*   `task`: Contains classes that define the different types of tasks that an NPC can have.
*   `transaction`: Contains classes related to objective transactions.
