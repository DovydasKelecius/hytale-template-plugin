# com.hypixel.hytale.builtin.adventure.worldlocationcondition

This package defines conditions based on world locations.

## Overview

The `worldlocationcondition` package provides a system for creating conditions that are based on the player's location in the world. It includes a plugin class for the feature and a class for defining a neighbor block tag condition.

### Key Classes

*   `WorldLocationConditionPlugin`: The main plugin class for the world location condition feature.
*   `NeighbourBlockTagsLocationCondition`: A condition that checks the tags of neighboring blocks.
