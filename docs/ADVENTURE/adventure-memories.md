# com.hypixel.hytale.builtin.adventure.memories

This package manages player memories and related gameplay mechanics.

## Overview

The `memories` package provides a system for storing and retrieving player memories, which can be used to track player progress and trigger events. It includes commands for managing memories, as well as a UI for displaying them to the player.

### Key Classes

*   `MemoriesPlugin`: The main plugin class for the memories feature.
*   `MemoriesCommand`: A command that allows players to manage their memories.
*   `PlayerMemories`: A component that stores a player's memories.
*   `MemoriesWindow`: A UI window that displays a player's memories.

### Sub-packages

*   `commands`: Contains the `MemoriesCommand` class.
*   `component`: Contains the `PlayerMemories` component.
*   `interactions`: Contains interactions related to memories.
*   `memories`: Contains the `Memory` class and related classes.
*   `page`: Contains the `MemoriesPage` class, which is a UI page for displaying memories.
*   `temple`: Contains features related to the Forgotten Temple.
*   `window`: Contains the `MemoriesWindow` class.
