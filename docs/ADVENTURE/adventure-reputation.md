# com.hypixel.hytale.builtin.adventure.reputation

This package provides a general-purpose reputation system.

## Overview

The `reputation` package provides a system for managing player reputation with different factions or groups. It includes commands for managing reputation, as well as a UI for displaying it to the player.

### Key Classes

*   `ReputationPlugin`: The main plugin class for the reputation feature.
*   `ReputationCommand`: A command that allows players to manage their reputation.
*   `ReputationGroupComponent`: A component that stores a player's reputation with a specific group.
*   `ReputationDataResource`: A resource that stores all reputation data.

### Sub-packages

*   `assets`: Contains asset definitions for reputation.
*   `choices`: Contains classes related to reputation-based choices.
*   `command`: Contains the `ReputationCommand` class and other reputation-related commands.
*   `store`: Contains the `ReputationDataResource` class.
