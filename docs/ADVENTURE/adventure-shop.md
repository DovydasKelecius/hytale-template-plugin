# com.hypixel.hytale.builtin.adventure.shop

This package provides a general-purpose shop system.

## Overview

The `shop` package provides a system for creating and managing shops. It includes a plugin class for the feature and classes for defining shop assets and UI.

### Key Classes

*   `ShopPlugin`: The main plugin class for the shop feature.
*   `ShopAsset`: An asset that defines a shop.
*   `ShopPage`: A UI page that displays a shop's inventory.

### Sub-packages

*   `barter`: Contains classes related to bartering.
