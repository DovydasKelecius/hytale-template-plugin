# com.hypixel.hytale.builtin.adventure.objectivereputation

This package manages reputation changes based on objective completion.

## Overview

The `objectivereputation` package provides a system for awarding reputation to players when they complete objectives. It includes a plugin class for the feature and classes for defining reputation rewards.

### Key Classes

*   `ObjectiveReputationPlugin`: The main plugin class for the objective reputation feature.
*   `ReputationCompletion`: A class that represents a reputation reward for completing an objective.
*   `ReputationCompletionAsset`: An asset that defines a reputation reward.

### Sub-packages

*   `assets`: Contains asset definitions for reputation rewards.
*   `historydata`: Contains classes for storing historical data about reputation rewards.
