# com.hypixel.hytale.builtin.adventure.npcshop

This package contains functionality for NPC-driven shops.

## Overview

The `npcshop` package provides a system for creating and managing shops that are run by NPCs. It includes a plugin class for the feature and a sub-package for NPC-specific shop behavior.

### Key Classes

*   `NPCShopPlugin`: The main plugin class for the NPC shop feature.

### Sub-packages

*   `npc`: Contains classes related to NPC behavior in shops.
